# Geometric-Survivor
Sopravvivi all'orda! In questo frenetico rogue-lite al neon, schiva infiniti nemici geometrici mentre spari in automatico. Raccogli cristalli, sali di livello e crea build devastanti con razzi, laser e droni. Sconfiggi i Boss per ottenere casse epiche, sblocca nuovi eroi e potenzia l'equipaggiamento. Domina l'arena!

link per giocare: https://include-nico.github.io/Geometric-Survivor/
